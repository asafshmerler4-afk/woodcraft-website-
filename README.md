# מוקו ושות׳ | Moku & Co.

אתר MVP לעסק ערכות נגרות DIY וסדנאות לבתי ספר.

A DIY woodworking kit business website with school workshops, built with React + Vite + Tailwind CSS.

## Tech Stack

- **React 18** — UI framework
- **Vite** — build tool
- **Tailwind CSS** — utility-first styling
- **Lucide React** — icons

## Features

- 🌐 Bilingual: Hebrew (RTL) / English (LTR) — toggle in navbar
- 📱 Fully responsive (mobile + desktop)
- 🪵 Japandi aesthetic — warm wood tones + deep green
- 🧩 Component-based architecture

## Getting Started

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```
